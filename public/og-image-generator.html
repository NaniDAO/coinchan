To generate the PNG:

Option 1 (Chrome DevTools):

  1. Open Chrome DevTools (F12)
  2. Press Ctrl+Shift+P (Cmd+Shift+P on Mac)
  3. Type "screenshot" and select "Capture full size screenshot"
  4. Crop to exactly 1200×630 if needed
  5. Save as og-image.png in the public/ folder

Option 2 (Screenshot tool):

  1. Zoom to 100%
  2. Use a screenshot tool to capture exactly the top 1200×630px area
  3. Save as og-image.png

Option 3 (Automated - recommended):

Use a headless browser tool like Playwright or Puppeteer to capture at exact dimensions.
